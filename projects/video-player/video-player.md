# Video Player PC Software

**Category:** PC Software
**Role:** Freelancer

## Overview
A custom-built desktop application designed for educational purposes, focusing on seamless media playback combined with student-centric progressive tracking algorithms.

## Features
- Developed software specifically for managing educational course videos.
- Created algorithms allowing users to save their exact viewing progress and associate timestamped text notes with each video.
- Designed a clean, intuitive, and user-friendly GUI.
- Built a robust local database backend using Visual Studio to securely track user progress over time.
- Programmed entirely in **C#** utilizing the .NET framework.
