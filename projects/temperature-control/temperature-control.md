# Temperature Control System for a Cosmetic Medical Device Furnace

**Category:** Embedded Control, Hardware & PCB
**Role:** Freelance Hardware / Firmware Developer

## Overview
This freelance project required building an extremely precise thermal environment tailored for cosmetic medical applications. The hardware and software were built from the ground up to ensure patient safety and operation consistency.

## Features
- Precise temperature control utilizing closed-loop **PID** tracking.
- Designed and programmed a custom printed circuit board (**PCB**) for the core logic and sensor interfaces.
- Designed an intuitive **NEXTION HMI** (Human-Machine Interface) touchscreen panel and programmed its UART communications to control the furnace natively.
