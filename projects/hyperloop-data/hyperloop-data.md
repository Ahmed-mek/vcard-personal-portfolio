# Hyperloop Data Transmission System

**Category:** Embedded Control, PC Software
**Role:** Embedded Engineer
**Event:** Hyperloop Development Competition 🇹🇷 (2023)

## Overview
Participated in building prototypes for the futuristic Hyperloop transportation concept. My distinct responsibility within the engineering framework was designing a high-speed, zero-loss digital communication web between the traveling pod and stationary ground control.

## Technical Details
- Designed and physically implemented a sophisticated data transmission system enabling continuous, real-time metrics passing from the train hull to the monitoring station.
- Selected and programmed an **STM32F7** microcontroller relying on its native high-speed Hardware Ethernet capabilities to interface perfectly and asynchronously transmit data to our internal router assembly.
- Designed the network topology so the mobile router transmitted data via high-bandwidth Wi-Fi to be successfully harvested and unpacked by a remote computer unit using strict **TCP protocols**.
