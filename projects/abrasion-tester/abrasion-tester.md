# Control Board Design for Abrasion Tester Machine

**Category:** Embedded Control, Hardware & PCB
**Role:** Graduation Project / R&D Engineer

## Overview
This project focused on the complete control system design for an industrial Abrasion Tester Machine, bridging the gap between high-power electro-mechanical actuation and precise digital user control.

## Technical Details
- Designed an industrial schematic and PCB for a **DC motor driver** explicitly tailored to the tester's torque needs.
- Implemented **PID** algorithms for closed-loop, highly responsive DC motor speed control, vital for consistent testing conditions.
- Engineered circuits to process and filter noisy analog and digital signals originating from the harsh testing environment.
- Developed a complete **PC user interface** using **C#** for real-time monitoring and parameter configuration.
