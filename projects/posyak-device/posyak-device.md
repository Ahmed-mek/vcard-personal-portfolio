# POSYAK Assistive Device

**Category:** Embedded Control
**Role:** Embedded Developer
**Event:** Barrier-Free Living Technologies Competition 🇹🇷 (2022)

## Overview
Engineered **POSYAK**, an innovative internet-of-things (IoT) medical device developed from the ground up specifically for patients battling Parkinson's disease.

## Features
- Hardware incorporates gyroscopic and accelerometer sensors to constantly map the patient's head tilt.
- Designed logic states where the device physically sends an alert (vibration/sound) when the patient’s neck remains dangerously bent beyond a safe set duration limits to prevent posture complications.
- Successfully integrated a Bluetooth/Wi-Fi stack allowing it to speak directly with a companion mobile app.
- Capable of recording and mathematically processing raw head movement data for a physician's later review.
