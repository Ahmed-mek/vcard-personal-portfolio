# Temperature Control System

**Category:** Hardware & PCB, PC Software

## Overview
This project involves designing a robust Temperature Control System using advanced PID algorithms on an STM32 microcontroller.

## Features
- Highly accurate temperature reading (`±0.1°C`).
- Custom developed PCB with isolated power supplies.
- Real-time data monitoring via PC application over UART.
